## bite-skill

Skill template for /bite. Takes comma-separated chomp names and an intent string. Formulates 3-8 research questions collaboratively with the user, chunks each chomp file, runs rlm-subcall subagents against all chunks, then synthesizes research.md (narrative findings) and surface-map.md (API mapping table).

**Source:** `.claude/skills/chomp-init/templates/bite-skill.md`
