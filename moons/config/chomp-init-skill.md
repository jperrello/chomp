## chomp-init-skill

Skill definition for /chomp-init. Creates chomp/ directories, copies chomp and bite skills plus the rlm-subcall agent into the project's .claude/ directory, appends chomp instructions to CLAUDE.md, updates .gitignore, and optionally runs an initial `chomp local` dump if the repo has enough files.

**Source:** `.claude/skills/chomp-init/SKILL.md`
